# Rock Paper Scissors Game

A simple web-based Rock Paper Scissors game built with HTML, CSS, and JavaScript.

## Features

- Play against the computer
- Choose rock, paper, or scissors
- View game results and score tracking
- Reset score functionality

## How to Run

1. Open `index.html` in your web browser.
2. Click on the rock, paper, or scissors buttons to make your choice.
3. The game will display the result and update the score.

## Files

- `index.html`: Main HTML file
- `script.js`: JavaScript logic for the game
- `style.css`: Styling for the game interface
- `images/`: Folder containing game images (rock.jpg, paper.png, scissor2.png)

## Troubleshooting

- Ensure all files are in the same directory.
- Make sure your browser supports JavaScript.
- If images don't load, check the `images/` folder path.
